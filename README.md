# UNITEC-TTB
Proyecto de implementación un juego interactivo entre persona hardware.
El proposito del juego es recrear un juego donde podamos interactuar mediante componentes eléctricos, nuestra implementación se basa en manipular micrófono, bocinas, botones, palanca y led's por meio de un microcontrolador como el ESP32-S3.
Dentro del repositorio se encuentran las carpetas que conforman las especificaciones de nuestro proyecto a realizar.
